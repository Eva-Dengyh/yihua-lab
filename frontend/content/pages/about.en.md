## Research Projects

### Py-Pose-Web: Intelligent Web Platform Based on Human Pose Estimation

A system integrating deep learning with full-stack architecture, implementing human skeletal keypoint detection based on Ultralytics YOLOv8, with modular backend interfaces built through Flask Blueprints.

**Tech Stack:** YOLOv8 Pose, Flask, Blueprints, JavaScript, OpenCV, Numpy, HTML/CSS, PostgreSQL

### Face-VectorDB-Login: Face Recognition Login System Based on Vector Database

A facial identity authentication system based on vector retrieval mechanisms, using RetinaFace and ArcFace for high-precision face detection and recognition, combined with FAISS for efficient feature vector matching.

**Tech Stack:** RetinaFace, ArcFace, iResNet50, FAISS, Flask, Electron, React, Spring Boot, PostgreSQL

### Label-Explorer: Multi-source Data Annotation and Visual Exploration Tool

A dataset analysis platform designed for YOLO format, built as a cross-platform desktop application with Electron, supporting unified management and visualization of raw images, labels, categories, and metadata files.

**Tech Stack:** Electron, Node.js, Python, Label Studio API

### Deep Learning-based Face Video Conference Check-in System

A complete video conference attendance management platform integrating video communication with intelligent face recognition. Frontend built with React + Electron, backend powered by Spring Boot with MySQL.

**Tech Stack:** React, Electron, Spring Boot, MySQL, Ubuntu, DJL, RetinaFace, ArcFace, Jitsi

### Blockchain Smart Contract Development and Application System

A decentralized registration and verification platform based on blockchain technology for digital content copyright protection, implementing on-chain asset authentication through Solidity smart contracts.

**Tech Stack:** Solidity, Ethereum, Web3, Linux, Docker
