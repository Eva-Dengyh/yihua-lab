## About Me

Hello! I'm Jeremy Bentham, a software developer and blogger.

This blog is built with [Next.js](https://nextjs.org) and styled with [Tailwind CSS](https://tailwindcss.com), based on the elegant [Chic](https://github.com/Siricee/hexo-theme-Chic) theme.

## About This Site

This site features:

- Clean, minimal design with focus on readability
- Dark mode support
- Responsive layout for all devices
- Code syntax highlighting
- Table of contents for long articles
- Tag and category organization
